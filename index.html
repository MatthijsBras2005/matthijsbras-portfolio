Projecten

Stauder Technologies

Voor het vak Project Reporting werk ik met mijn projectgroep aan een webapp voor Stauder Technologies Europe. Eerst hebben we onderzoek gedaan, requirements uitgewerkt en een adviesrapport geschreven met mogelijke oplossingsrichtingen.

Nu bouwen we de gekozen oplossing uit tot een app-prototype. Ik werk mee aan de backend, requirements, appstructuur en dashboardconcepten. Daarnaast heb ik de volledige UX/UI van de applicatie geoptimaliseerd om de gebruiksvriendelijkheid te maximaliseren.

Vanwege NDA blijven interne data, screenshots en vertrouwelijke details buiten beeld.

Reisbureau Oracle

In dit groepsproject hebben we een inkoopsysteem gebouwd in Oracle APEX, inclusief leveranciersmanagement voor verschillende reisonderdelen.

Als hoofd UX'er van mijn groep was ik verantwoordelijk voor de afstemming van de UX en UI aspecten van de applicatie met de UX'ers van de andere groepen.

Daarnaast heb ik voor dit project uitvoerig software getest, waaronder Use Case testen, Procescyclustesten, usability testing en exploratory testing.
